# 好游 - 旅行路线规划应用 使用说明

## 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 配置高德地图API
在 `index.html` 文件中，找到以下代码并替换为你的高德地图API密钥：

```html
<script type="text/javascript">
  window._AMapSecurityConfig = {
    securityJsCode: '你的安全密钥',
  }
</script>
<script src="https://webapi.amap.com/maps?v=2.0&key=你的高德地图Key"></script>
```

**获取高德地图API Key步骤：**
1. 访问 [高德开放平台](https://lbs.amap.com/)
2. 注册并登录账号
3. 进入"控制台" → "应用管理" → "我的应用"
4. 创建新应用，添加Web端(JS API)服务
5. 获取Key和安全密钥

### 3. 启动开发服务器
```bash
npm run dev
```

然后在浏览器中打开显示的地址（通常是 http://localhost:5173）

## 功能介绍

### 📱 登录/注册系统

#### 登录页面
- **密码登录**：使用手机号+密码登录
- **验证码登录**：使用手机号+验证码登录（演示验证码：123456）
- 支持密码显示/隐藏切换
- 自动验证手机号格式（11位，1开头）

#### 注册页面
- 手机号验证
- 验证码发送（演示验证码：123456）
- 输入正确验证码后才能设置密码
- 两次密码确认
- 注册成功后自动登录

**演示账号信息：**
- 验证码：123456
- 密码：任意6位以上字符

### 🗺️ 地图功能

#### 1. Leaflet路线规划（首页）
- 输入起点和终点
- 点击"一键规划路线"查看地图
- 支持获取当前位置

#### 2. 高德地图（地点搜索）
点击首页的"高德地图"按钮进入，功能包括：

**搜索功能**
- 🔍 关键词搜索（地点、景点、餐厅等）
- 💡 智能搜索建议（自动完成）
- 🏷️ 分类快速搜索（景点、餐厅、酒店、购物、地铁）

**定位功能**
- 📍 定位当前位置
- 🏛️ 搜索附近景点（2km范围内）
- 🍽️ 探索附近餐饮（1.5km范围内）

**地图交互**
- 点击搜索结果在地图上标记
- 自动缩放到合适的视角
- 显示POI详细信息

### 👤 个人中心
- 显示当前登录手机号（脱敏显示）
- 退出登录功能

## 常见问题

### Q: 地图无法显示？
A: 请检查：
1. 是否配置了正确的高德地图API Key
2. 网络连接是否正常
3. 浏览器控制台是否有错误信息

### Q: 验证码登录/注册失败？
A: 这是演示版本，请使用固定验证码：**123456**

### Q: 如何清除登录状态？
A: 点击首页右上角的"退出"按钮，或清除浏览器localStorage

### Q: 是否支持真实的后端API？
A: 当前版本使用localStorage模拟，需要接入真实后端API请修改：
- `LoginPage.jsx` 中的 `handleSubmit` 函数
- `RegisterPage.jsx` 中的 `handleSubmit` 函数
