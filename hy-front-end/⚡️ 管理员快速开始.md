## 步骤1：注册管理员（30秒）

### 方法A：使用测试页面（推荐）✨

打开浏览器访问：
```
http://localhost:5173/test-admin-register.html
```

点击"注册管理员"按钮即可（默认账号已填好）

### 方法B：使用控制台

浏览器按 F12，复制执行：

```javascript
fetch('http://localhost:8081/api/auth/admin/quick-register', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    phone: "18888888888",
    password: "123123"
  })
}).then(res => res.json()).then(data => {
  if (data.code === 200) alert('✅ 注册成功！');
  console.log(data);
});
```
