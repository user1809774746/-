# 帖子审核系统 - 快速使用指南

## 🚀 快速开始

### 管理员登录

1. **使用管理员账号登录系统**
   - 账号类型：admin
   - 登录后自动进入"帖子审核管理系统"

2. **系统界面**
   ```
   ┌─────────────────────────────────────────┐
   │  帖子审核管理系统         [退出管理后台] │
   ├─────────────────────────────────────────┤
   │  [待审核帖子] [所有帖子] [统计信息]      │
   └─────────────────────────────────────────┘
   ```

---

## 📋 主要功能

### 1. 待审核帖子（最常用）

**在这里可以：**
- 查看所有等待审核的帖子
- 快速审核通过或拒绝
- 查看帖子详情

**操作步骤：**
```
1. 点击 [待审核帖子] 标签
2. 浏览帖子列表
3. 点击 [查看详情] 查看完整内容
4. 点击 [通过] 批准发布
   或
   点击 [拒绝] 输入拒绝原因
```

**审核通过后：**
- ✅ 帖子对所有用户可见
- ✅ 用户会看到"审核通过"提示

**审核拒绝后：**
- ❌ 帖子变为草稿状态
- ❌ 用户会看到拒绝原因
- ❌ 用户可以修改后重新提交

---

### 2. 所有帖子（高级管理）

**在这里可以：**
- 查看和管理所有帖子
- 筛选不同状态的帖子
- 设置精选和置顶
- 删除违规帖子

**筛选功能：**
```
审核状态：
- 全部
- 待审核
- 已通过
- 已拒绝

发布状态：
- 全部
- 草稿
- 已发布
- 已删除
```

**特殊功能：**（仅限已审核通过的帖子）
- 🌟 **设置精选**：优质帖子推荐
- 📌 **设置置顶**：重要帖子置顶显示

---

### 3. 统计信息（数据分析）

**查看统计数据：**
- 📊 待审核数量
- ✅ 已通过数量
- ❌ 已拒绝数量
- 📈 总帖子数
- 🌟 精选帖子数量
- 📌 置顶帖子数量

---

## 🎯 常见操作场景

### 场景1：审核新帖子

```
步骤：
1. 进入 [待审核帖子] 页面
2. 看到新提交的帖子
3. 点击 [查看详情] 阅读完整内容
4. 判断内容是否合规：
   
   ✅ 内容合规 → 点击 [通过]
   ❌ 内容违规 → 点击 [拒绝] → 输入原因 → 确认

5. 审核完成，帖子从待审核列表移除
```

### 场景2：设置精选帖子

```
步骤：
1. 进入 [所有帖子] 页面
2. 筛选"审核状态" = "已通过"
3. 找到优质帖子
4. 点击 [设为精选]
5. 完成！帖子将在精选区域展示
```

### 场景3：处理违规帖子

```
步骤：
1. 发现违规已发布的帖子
2. 点击 [删除] 按钮
3. 输入删除原因（可选）
4. 确认删除
5. 帖子状态变为"已删除"
```

### 场景4：查看审核数据

```
步骤：
1. 进入 [统计信息] 页面
2. 查看各项统计数据
3. 了解当前审核工作量
4. 根据数据调整审核策略
```

## 💡 审核建议

### 审核通过的标准
✅ 内容真实、原创  
✅ 语言文明、无违禁词  
✅ 信息准确、有价值  
✅ 图片清晰、无水印  
✅ 符合社区规范  

### 审核拒绝的原因
❌ 内容虚假、抄袭  
❌ 含有违禁信息  
❌ 广告营销内容  
❌ 低质量、无价值  
❌ 图片模糊、不当  

### 拒绝原因示例
```
好的拒绝原因（具体明确）：
- "帖子内容与标题不符，请修改标题或补充相关内容"
- "图片质量较差且含有其他平台水印，请更换高清无水印图片"
- "攻略信息已过时（2020年），请更新为最新信息"

不好的拒绝原因（模糊不清）：
- "内容不合格"
- "需要修改"
- "不通过"
```

## 🛠️ 故障排查

### 问题1：无法进入审核系统
**原因**：账号不是管理员  
**解决**：检查账号类型，确保使用admin账号登录

### 问题2：审核操作失败
**原因**：网络问题或权限不足  
**解决**：刷新页面，重新登录

### 问题3：看不到待审核帖子
**原因**：暂时没有待审核的帖子  
**解决**：这是正常情况，等待用户发布新帖子

### 问题4：拒绝操作无效
**原因**：未填写拒绝原因  
**解决**：必须填写拒绝原因才能提交
