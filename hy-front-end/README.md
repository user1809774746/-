# 路线规划助手 - FE-LBX-0.0.1

## 项目概述
这是一个基于 React + Vite 的移动端路线规划应用，使用 Tailwind CSS 进行样式设计，集成 Leaflet.js 地图组件。

## 技术栈
- **前端框架**: React 18
- **构建工具**: Vite 5
- **样式框架**: Tailwind CSS 3
- **地图组件**: Leaflet.js + React-Leaflet
- **开发语言**: JavaScript (JSX)

## 项目结构
```
├── index.html              # 入口 HTML 文件
├── package.json            # 项目依赖配置
├── vite.config.js          # Vite 配置
├── tailwind.config.js      # Tailwind CSS 配置
├── postcss.config.js       # PostCSS 配置
└── src/
    ├── main.jsx            # 应用入口
    ├── App.jsx             # 主应用组件
    ├── index.css           # 全局样式
    └── components/
        ├── HomePage.jsx    # 首页组件
        └── MapPage.jsx     # 地图页面组件
```
