## 🔍 关键信息速查

### 新增的 API 接口

```
POST /api/post/draft/{draftId}/convert-and-publish  # 发布草稿
DELETE /api/post/draft/{draftId}                    # 删除草稿
```

### 前端必须修改的地方

1. ✅ 保存草稿时包含 `draftData` 字段
2. ✅ 发布草稿使用新接口 `convert-and-publish`
3. ✅ 删除草稿使用新接口 `DELETE`

### 数据库变更

- ✅ `user_travel_post` 表移除了帖子内容字段
- ✅ 采用双表架构：`travel_post` + `user_travel_post`
